Projet E5

Développement d'un Back-end et d'un Front-end

Au cours de ma deuxième année de BTS SIO option SLAM au sein de l'EPSI Lyon, il m'a été donné l'opportunité de développer un projet liant une partie back-end et une partie front-end. Le but de ce projet était de créer une application web permettant de récupérer la liste des départements français dans une base de données et de créer une deuxième base de données dans laquelle il serait possible d'ajouter, de modifier ou de supprimer les coordonnées des médecins. Un utilisateur pourrait ainsi chercher les coordonnées des médecins présents dans son département.



Afin de pouvoir accéder à la liste des départements, il suffit de cliquer sur le bouton "Départements" en haut de la page. Nous arrivons ensuite sur la liste des départements où nous pouvons cliquer sur détails afin d'obtenir les coordonnées des médecins présents dans le département demandé, comme le montre l'exemple ci-dessous.




Ensuite, j'ai créé la page internet permettant d'afficher les coordonnées de tout les médecins inscrits sur l'application. Il est également possible pour un médecin connecté sur l'application de modifier ses coordonnées, de les supprimer ou encore de créer son profil.



Cependant, afin de pouvoir modifier ces données, il faut être connecté et posséder un token devant être généré au pré alable.
